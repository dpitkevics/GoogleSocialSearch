{% trans 'Set as homepage' %}

{% trans 'Google Chrome' %}

  1. {% blocktrans %} In the top-right corner of your window, click the Chrome menu {% endblocktrans %}
  2. {% blocktrans %} Select Settings. (If you're using a Chromebook, click "Show advanced settings" at the bottom of the page.) {% endblocktrans %}
  3. {% blocktrans %} Under "On startup," click Open a specific page or set of pages. {% endblocktrans %}
  4. {% blocktrans %} Click Set pages. {% endblocktrans %}
  5. {% blocktrans %} Enter jooglin.com in input field. {% endblocktrans %}
  6. {% blocktrans %} Click OK. {% endblocktrans %}
  7. {% blocktrans %} To retrieve more info, Click Here {% endblocktrans %}

{% trans 'Mozilla Firefox' %}

  1. {% blocktrans %} Open a tab with the web page you want to use as your home page. {% endblocktrans %}
  2. {% blocktrans %} Drag and drop that tab onto the Home button Home Button . {% endblocktrans %}
  3. {% blocktrans %} Click Yes to set this page as your home page. {% endblocktrans %}
  4. {% blocktrans %} To retrieve more info, Click Here {% endblocktrans %}

{% trans 'Internet Explorer' %}

  1. {% blocktrans %} On the Start screen, tap or click Internet Explorer to open Internet Explorer. {% endblocktrans %}
  2. {% blocktrans %} Go to a page that you'd like to set as a home page. {% endblocktrans %}
  3. {% blocktrans %} Swipe in from the right edge of the screen, and then tap Settings.
    (If you're using a mouse, point to the lower-right corner of the screen, move the mouse pointer up, and then click Settings.) {% endblocktrans %}
  4. {% blocktrans %} Tap or click Options, and under Home pages, tap or click Customize. {% endblocktrans %}
  5. {% blocktrans %} Tap or click Add current site, edit the URL if you'd like, and then tap or click Add to make the site one of your home pages. {% endblocktrans %}
  6. {% blocktrans %} To retrieve more info, Click Here {% endblocktrans %}

{% trans 'Safari' %}

  1. {% blocktrans %} Choose Safari > Preferences, then click General. {% endblocktrans %}
  2. {% blocktrans %} Enter a webpage address, which is http://jooglin.com in the Homepage field {% endblocktrans %}
  3. {% blocktrans %} To retrieve more info, Click Here {% endblocktrans %}

{% trans 'Opera' %}

  1. {% blocktrans %} Open Opera > Preferences > General. {% endblocktrans %}
  2. {% blocktrans %} Enter a webpage address, which is http://jooglin.com in the Homepage field {% endblocktrans %}
  3. {% blocktrans %} To retrieve more info, Click Here {% endblocktrans %}
