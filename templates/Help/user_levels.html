{% trans 'User Levels' %}

  • {% blocktrans %} Basic Level - User have privileges to access his profile in Jooglin Search. This does not give permission to buy search items, comment search items or vote on search items. {% endblocktrans %}
  • {% blocktrans %} Novice Level - User has access to all main features of Jooglin Search - item buying and selling, commenting and voting, administrating items user owns. {% endblocktrans %}
  • {% blocktrans %} Advanced Level - User has all features that are on previous levels. Also, user can add basic HTML tags to their owned search items in "owner comment" field. These HTML tags include - a, abbr, acronym, b, blockquote, code, em, i, li, ol, strong, ul. {% endblocktrans %}

{% trans 'User Level Experiences' %}

  • {% blocktrans %} Basic Level: 0 - 500 {% endblocktrans %}
  • {% blocktrans %} Novice Level: 501 - 2000 {% endblocktrans %}
  • {% blocktrans %} Advanced Level: 2001 - 10000 {% endblocktrans %}

{% trans 'Experience Receiving' %}

{% blocktrans %} User receives experience by using Jooglin Search - making searches, voting on search results, commenting them, buying and selling search results. {% endblocktrans %}
