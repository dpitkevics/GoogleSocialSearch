{% trans 'Set as default search engine' %}

{% trans 'Google Chrome' %}

  1. {% blocktrans %} In the top-right corner of your window, click the Chrome menu {% endblocktrans %}
  2. {% blocktrans %} Select Settings. (If you're using a Chromebook, click "Show advanced settings" at the bottom of the page.) {% endblocktrans %}
  3. {% blocktrans %} Go to the "Search" section. {% endblocktrans %}
  4. {% blocktrans %} Click Manage search engines.... {% endblocktrans %}
  5. {% blocktrans %} Scroll to very bottom of new modal window. {% endblocktrans %}
  6. {% blocktrans %} In Add new search engine field enter Jooglin.com. {% endblocktrans %}
  7. {% blocktrans %} In Keyword field enter Jooglin.com. {% endblocktrans %}
  8. {% blocktrans %} In URL with %s in place of query field enter http://jooglin.com/?query=%s. {% endblocktrans %}
  9. {% blocktrans %} Click outside of input box and see that new search engine is added. {% endblocktrans %}
  10. {% blocktrans %} Find Jooglin.com search engine (Jooglin.com should appear in First column), point mouse over it and click Make default {% endblocktrans %}
  11. {% blocktrans %} Click Done {% endblocktrans %}
  12. {% blocktrans %} To retrieve more info, Click Here {% endblocktrans %}
